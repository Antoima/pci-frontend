# PCI-Frontend

## Descripción del Proyecto

PCI-Frontend es una aplicación web diseñada para gestionar campañas de marketing con influencers. Permite a los usuarios proyectar el alcance de sus campañas, seleccionar influencers y ajustar detalles del brief.

## Instalación

Sigue estos pasos para instalar y configurar el proyecto en tu máquina local.

1. Clona el repositorio:
   ```sh
   git clone https://github.com/tu-usuario/pci-frontend.git
   ```
2. Navega al directorio del proyecto:
   ```sh
   cd pci-frontend
   ```
3. Instala las dependencias:
   ```sh
   npm install
   ```

## Uso

Para iniciar la aplicación, tienes dos opciones:

1. Abre el archivo [index.html](http://_vscodecontentref_/2) directamente en tu navegador haciendo doble clic sobre él.
2. Ejecuta un servidor local con Python:
   ```sh
   python -m http.server 8000
   ```
   Luego, abre tu navegador y navega a `http://localhost:8000`.
